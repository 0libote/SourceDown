# SourceDown roadmap

## Conversion engines

- Improve Docling and Marker support and converter recommendations.
- Retry a failed or poor conversion with another converter.
- Add conversion comparison mode.
- Consider MinerU for OCR-heavy, scientific, and formula-heavy PDFs.
- Consider Pandoc for clean DOCX, HTML, EPUB, RTF, and ODT conversion.

## Docling

- Expand conversion across DOCX, PPTX, XLSX, HTML, EPUB, images, EML, MSG, plain text, Markdown-like files, audio, and ODF.
- Improve OCR for scanned PDFs and images.
- Add structured JSON export for richer metadata and AI workflows.
- Improve tables, charts, images, and reading order.
- Improve EML and MSG email imports.
- Add audio transcript imports where useful.

## Marker

- Expand conversion across PDF, images, PPTX, DOCX, XLSX, HTML, and EPUB.
- Add page ranges, table-only extraction, and OCR-only conversion.
- Add JSON, HTML, and chunk export modes.
- Improve forms, equations, images, tables, and technical documents.
- Add optional LLM cleanup as an advanced setting.
